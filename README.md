# PCOS Risk Prediction using Machine Learning

## Project Overview
This project predicts the risk of Polycystic Ovary Syndrome (PCOS) using a Machine Learning model.

## Features
- Data preprocessing
- Random Forest Classifier
- Probability-based risk categorization
- Model saving using joblib

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## How to Run

1. Install requirements:
pip install -r requirements.txt

2. Train model:
python src/train.py

3. Predict:
python src/predict.py
